# Continuous detection and containment qualification

W9 combines live production signal coverage, remote immutable telemetry with
bounded lag, monitoring-failure alerts, production pager delivery, live human
acknowledgement and escalation, ordered automated containment, digest-bound
known-clean destructive restore, independent signed red-team closure/retest,
and the production evidence plane.

The gate rejects heartbeat stubs, mock webhooks, example rosters, missed or late
acknowledgements, partial/reordered containment, dirty restore artifacts,
same-organization red teams, open findings, failed retests, and local or replayed
evidence.

Independent periodic red-team evidence contributes to E4 but is not continuous
independent E5 verification. W9 remains pending until live operations produce
the required receipts.
